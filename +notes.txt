Divergence from JavaScript
  Cannot Implement:
    "0" -> truthy
    arguments.caller
  Not Implemented:
    function.length
    Implicitly setting globals
    eval or new Function(string)

Compiling:
  keyword literals: null -> _Null::$null, undefined -> null
  {a: 1} -> new _Object('a', 1)
  [1, 2] -> new _Array(1, 2)
  function () {} -> new _Function(function() use (...) {})
  /abc/i -> new _RegExp('abc', 'i')
  operators to functions (add, typeof, instance_of, or_, and_, eq)
  compute a function's `use` (lexical scope) at compile time

Optimisations:
  functions can be instantiated using $Array->construct()
